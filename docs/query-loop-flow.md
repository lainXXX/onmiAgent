# Query Loop 执行流程

> Agent 核心循环的完整执行流程

---

## 1. 整体流程图

```
┌──────────────────────────────────────────────────────────────────────────┐
│                         Query Loop 完整流程                               │
└──────────────────────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────────────┐
    │                         用户输入                                  │
    │                     User Input / Message                          │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    HOOK: SessionStart                            │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                  HOOK: UserPromptSubmit                          │
    │                  (可取消、可修改输入)                              │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │              ========== 压缩流水线 ==========                     │
    │                                                                  │
    │    Layer 1 ──▶ Layer 2 ──▶ Layer 3 ──▶ Layer 4                  │
    │   SnipCompact  Micro  Collapse  AutoCompact                      │
    │       │          │          │          │                         │
    │   每轮必然   每轮必然   按需      按需                             │
    │                                                                  │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    HOOK: PreCompact                             │
    │                  (可取消、可追加指令)                              │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    调用 Claude API                                │
    │                    callModel(messages)                           │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    HOOK: Elicitation                             │
    │                  (可选：修改/拦截响应)                            │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    获取 Assistant Message                         │
    │                    (可能包含 tool_use)                           │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    检查 tool_use                                 │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                    ┌─────────────┴─────────────┐
                    │                           │
                   YES                         NO
                    │                           │
                    ▼                           ▼
    ┌───────────────────────────┐    ┌───────────────────────────┐
    │   执行工具编排              │    │     渲染响应               │
    │   runTools(tool_use)      │    │     返回给用户              │
    └───────────────────────────┘    └───────────────────────────┘
                    │
                    ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │              ========== 工具执行循环 ==========                  │
    │                                                                  │
    │    for each tool in tool_use:                                    │
    │        │                                                         │
    │        ├── HOOK: PreToolUse (可取消、可修改)                      │
    │        ├── HOOK: PermissionRequest                               │
    │        │    │                                                   │
    │        │    ├── 拒绝 ──▶ HOOK: PermissionDenied                 │
    │        │    │         └── HOOK: PostToolUseFailure              │
    │        │    │                                                   │
    │        │    └── 允许 ──▶ Tool.call()                            │
    │        │              ├── 成功 ──▶ HOOK: PostToolUse            │
    │        │              │         └── HOOK: StopHook              │
    │        │              │                                           │
    │        │              └── 失败 ──▶ HOOK: PostToolUseFailure      │
    │        │                                                         │
    │        └── 返回工具结果 ──▶ 追加到 messages                       │
    │                                                                  │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │              工具结果返回 ──▶ 继续循环                            │
    │              (回到压缩流水线检查)                                 │
    │                                                                  │
    │              循环直到：无 tool_use 或达到最大迭代                   │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    HOOK: SessionEnd                              │
    │                    (会话结束)                                     │
    └─────────────────────────────┬───────────────────────────────────┘
                                  │
                                  ▼
                          等待下一轮输入
```

## 2. 详细阶段说明

### 2.1 阶段 1: 输入处理

```
用户输入
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  HOOK: SessionStart                                             │
│  - 初始化会话状态                                                │
│  - 记录开始时间                                                  │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  HOOK: UserPromptSubmit                                         │
│  - 验证输入                                                      │
│  - 清理/规范化                                                   │
│  - 可取消（返回错误）                                             │
│  - 可修改（替换输入）                                             │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  submitMessage(input)                                           │
│  - 追加用户消息到 messages[]                                     │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 阶段 2: 压缩流水线

```
压缩流水线入口
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 1: SnipCompact                                           │
│  - 去重                                                         │
│  - 截断超长工具输出                                               │
│  - 删除空消息                                                    │
│  - 合并相似消息                                                  │
│  - ⚠️ 每轮必然执行                                                │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 2: MicroCompact                                          │
│  - 时间衰减检查（gap > 60分钟?）                                   │
│  - 缓存编辑检查（工具结果数量 > 阈值?）                              │
│  - 清理旧工具结果                                                │
│  - ⚠️ 每轮必然执行（但可能跳过）                                    │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 3: Context Collapse                                      │
│  - Token 使用量检查                                              │
│  - < 90%: 跳过                                                   │
│  - 90%: Commit 模式（折叠视图）                                   │
│  - > 95%: Block 模式（阻塞）                                      │
│  - ⚠️ 按需执行                                                   │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  Layer 4: AutoCompact                                           │
│  - 阈值检查（tokens > contextWindow - 13K）                      │
│  - PreCompact Hook                                              │
│  - 摘要生成                                                      │
│  - PostCompact Hook                                             │
│  - ⚠️ 按需执行                                                   │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
  压缩完成
```

### 2.3 阶段 3: API 调用

```
压缩后的 messages[]
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  HOOK: PreCompact (最后检查点)                                    │
│  - 可取消压缩                                                    │
│  - 可追加自定义指令到 prompt                                       │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  callModel(messages)                                             │
│  - 构建 API 请求                                                 │
│  - 发送流式请求                                                  │
│  - 处理响应                                                      │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  HOOK: Elicitation                                              │
│  - 模型主动请求信息（可选）                                        │
│  - 可修改响应内容                                                │
│  - 可跳过（使用默认值）                                           │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  AssistantMessage                                               │
│  - content: 文本响应                                             │
│  - tool_use: 工具调用列表（可选）                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 阶段 4: 工具执行

```
有 tool_use?
    │
    ├── NO ──▶ 渲染响应 ──▶ 返回用户
    │
    └── YES ──▶ 进入工具编排
                   │
                   ▼
    ┌─────────────────────────────────────────────────────────────┐
    │  工具分区 (Concurrency-safe batching)                         │
    │  - 可并行执行的工具分组                                       │
    │  - 依赖顺序的执行分组                                         │
    └─────────────────────────────────────────────────────────────┘
                   │
                   ▼
    ┌─────────────────────────────────────────────────────────────┐
    │  遍历每个工具                                               │
    └─────────────────────────────────────────────────────────────┘
                   │
         ┌─────────┴─────────┐
         │                     │
    ┌────▼────┐          ┌────▼────┐
    │  工具 1  │          │  工具 2  │
    └────┬────┘          └────┬────┘
         │                     │
         ▼                     ▼
    ┌─────────────────────────────────────────────────────────────┐
    │  HOOK: PreToolUse                                               │
    │  - 可取消工具调用                                               │
    │  - 可修改工具输入                                               │
    └─────────────────────────────────────────────────────────────┘
         │                     │
         ▼                     ▼
    ┌─────────────────────────────────────────────────────────────┐
    │  HOOK: PermissionRequest                                    │
    │  - 检查用户授权                                                │
    │  - 检查安全策略                                                │
    └─────────────────────────────────────────────────────────────┘
         │                     │
    ┌────┴────┐          ┌────┴────┐
    │  拒绝   │          │   允许  │
    └────┬────┘          └────┬────┘
         │                     │
         ▼                     ▼
    ┌─────────────┐      ┌─────────────────────────────────────────┐
    │ HOOK:      │      │  Tool.call()                             │
    │ Permission │      │                                           │
    │ Denied    │      │  ┌─────────────────────────────────────┐  │
    │            │      │  │ BashTool: 执行 shell 命令           │  │
    │ HOOK:      │      │  │ FileReadTool: 读取文件              │  │
    │ PostTool   │      │  │ GrepTool: 搜索代码                  │  │
    │ UseFailure │      │  │ AgentTool: Spawn sub-agent          │  │
    └─────────────┘      │  │ ...                                  │  │
                         │  └─────────────────────────────────────┘  │
                         │         │                               │
                         │    ┌────┴────┐                         │
                         │    │         │                         │
                         │    ▼         ▼                         │
                         │ ┌────┐   ┌────┐                       │
                         │ │成功│   │失败│                       │
                         │ └────┘   └────┘                       │
                         │    │         │                         │
                         │    ▼         ▼                         │
                         │ HOOK:    HOOK:                         │
                         │ PostTool PostTool                      │
                         │ Use     UseFailure                    │
                         │         │                             │
                         │         ▼                             │
                         │    错误处理                            │
                         └─────────────────────────────────────────┘
                                   │
                                   ▼
                         ┌─────────────────────────────────────────┐
                         │  renderToolResultMessage()               │
                         │  - 格式化工具结果                        │
                         │  - 追加到 messages[]                      │
                         └─────────────────────────────────────────┘
                                   │
                                   ▼
                         ┌─────────────────────────────────────────┐
                         │  工具结果 ──▶ 回到压缩流水线检查          │
                         │  (循环直到无 tool_use)                    │
                         └─────────────────────────────────────────┘
```

## 3. 循环终止条件

```
Query Loop 可能在以下情况终止：

1. 正常完成
   - 无 tool_use 需要执行
   - 返回最终响应给用户

2. 达到最大迭代
   - 防止无限循环
   - 通常限制为 100-200 轮

3. 错误终止
   - API 错误（可重试）
   - 工具执行错误（不可恢复）
   - 用户取消

4. 资源限制
   - Token 超出最大限制
   - 内存超出限制
```

## 4. 消息流向

```
┌─────────────────────────────────────────────────────────────────┐
│                        messages[] 数组                           │
│                                                                 │
│  [0]  SystemMessage (系统提示)                                   │
│  [1]  UserMessage (用户消息)                                     │
│  [2]  AssistantMessage (AI 响应)                                 │
│  [3]  AssistantMessage (tool_use) ──┐                          │
│  [4]  UserMessage (工具结果) ─────────┼─▶ 工具调用循环          │
│  [5]  AssistantMessage (tool_use) ──┘                          │
│  [6]  AssistantMessage (最终响应)                               │
│  ...                                                            │
│                                                                 │
│  压缩后：                                                        │
│  [0]  SystemMessage                                             │
│  [1]  UserMessage (用户消息)                                     │
│  [2]  <!-- compact boundary -->                                  │
│  [3]  <summary> 压缩摘要 </summary>                              │
│  [4]  UserMessage (最近消息)                                     │
│  [5]  AssistantMessage (最终响应)                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 6. Hook 执行时机表

| Hook | 时机 | 可中断 | 可修改 |
|------|------|--------|--------|
| SessionStart | 会话开始 | ❌ | ❌ |
| UserPromptSubmit | 用户提交输入 | ✅ | ✅ |
| PreCompact | 压缩前 | ✅ | ✅ |
| Elicitation | 模型请求信息 | ❌ | ✅ |
| PreToolUse | 工具执行前 | ✅ | ✅ |
| PermissionRequest | 请求权限 | ❌ | ❌ |
| PermissionDenied | 权限拒绝 | ❌ | ❌ |
| PostToolUse | 工具成功 | ❌ | ❌ |
| PostToolUseFailure | 工具失败 | ❌ | ❌ |
| StopHook | 工具链结束 | ❌ | ✅ |
| PostCompact | 压缩后 | ❌ | ❌ |
| SessionEnd | 会话结束 | ❌ | ❌ |

## 7. 并发模型

```
工具执行策略：

1. 顺序执行 (Sequential)
   tool1 ──▶ tool2 ──▶ tool3

2. 并行执行 (Parallel)
   tool1 ──┐
   tool2 ──┼──▶ tool3
   tool3 ──┘

3. 依赖感知执行 (Dependency-aware)
   ┌─── tool1 ───┐
   │              │
   ├── tool2 ────┼──▶ tool5
   ├── tool3 ────┤
   │              │
   └── tool4 ────┘

通过工具分区 (Partitioning) 实现：
- 无依赖的工具并行
- 有依赖的工具顺序
- 保证工具结果正确引用
```

## 8. 错误恢复

```
┌─────────────────────────────────────────────────────────────────┐
│                       错误恢复策略                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  API 错误                                                       │
│  ├── 网络错误 ──▶ 自动重试 (3次指数退避)                          │
│  ├── 401/403 ──▶ 终止，不重试                                    │
│  ├── 404 ──▶ 终止，不重试                                        │
│  └── PROMPT_TOO_LONG ──▶ 触发 AutoCompact ──▶ 重试              │
│                                                                 │
│  工具错误                                                       │
│  ├── 可恢复 ──▶ 返回错误信息，继续执行                             │
│  └── 不可恢复 ──▶ 终止工具链                                     │
│                                                                 │
│  中断错误                                                       │
│  └── gracefulShutdown() ──▶ 保存状态 ──▶ 终止                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
